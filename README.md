# 🎬 Api de Consulta e Cadastro de Filmes

## Funcionalidades:
➕ Cadastrar novos filmes

🔍 Consultar filmes por ID ou título

📄 Listar todos os filmes cadastrados

🛠️ Tecnologias usadas

- Java 21

- Spring Boot

- Spring Data JPA

- H2 Database

- Springdoc OpenAPI (Swagger 3)

- Maven
